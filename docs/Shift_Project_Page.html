My work on project:

Shift

We did this project as an entry for a highschool STEM videogame contest and made it pretty far, but we found out that the contest had been cancelled.
The basic concept of the game was that the player had the ability to switch - or 'Shift' - between two different realities at will, with each realities having their own obstacles and such.
The player would need to use the concept of 'Shifting' to overcome puzzles and avoid getting caught by in game enemies. Our two realities would be that of a "Real" world and a "Virtual" world
The "Real" world would have more realistic graphics and higher poly count models set in a far dystopian future,
while the "Virtual" world would have low poly count models and older looking graphics set in the late 80s.

Here I'll touch on some of the things I did during the development period.

Character Logic

Once again I'll start with the character logic.
I was working on this before we decided to start development of Shift,
It's a consolidated logic set to make it easy to control the player from both a management
perspective and ease of use.
A complete package is available on the Unity Assets store for free here.
Of course this needed to be modified to work with our project, but most of that was done by my co-developer.

Shifting

I did the first version of the 'Shifting' mechanic, the concept and overall execution was simple.
But giving it that flare to make it look good was much harder.
I used a shader with vertex deformation to make a transition effect that I still am very happy with.

3D Modeling

I did all the 3d modeling for the project, I went with a lowpoly count style for most of it
and I'm pretty happy with the results. I learned many new methods for texturing and animating
in blender with all the props I made. I use a few in the demo scene for the linked asset package above.
Closer to the end of our development I made a level hub based in a city, it was much harder to create
than I thought it would be and ended up starting over many times, though I'm happy with the latest revision.
